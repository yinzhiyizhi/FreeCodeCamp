# Bootstrap: Call out Optional Actions with btn-info
Bootstrap comes with several pre-defined colors for buttons. The btn-info class is used to call attention to optional actions that the user can take.

Create a new block-level Bootstrap button below your "Like" button with the text "Info", and add Bootstrap's btn-info and btn-block classes to it.

Note that these buttons still need the btn and btn-block classes.


# 中文版本FCC
Bootstrap：使用 btn-info 调出可选操作

Bootstrap 有着丰富的预定义按钮颜色。浅蓝色 btn-infoclass 通常用在用户可能采取的操作上。

在你的 "Like" 按钮下方创建包含文本 "Info" 的块级 Bootstrap 按钮， 然后为该按钮添加 Bootstrap 的 btn-info和 btn-blockclass。

记住：你的这些按钮仍然需要 btn和 btn-blockclass。
