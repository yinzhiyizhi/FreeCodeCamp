# Bootstrap: Add Elements within Your Bootstrap Wells
Now we're several div elements deep on each column of our row. This is as deep as we'll need to go. Now we can add our button elements.

Nest three button elements within each of your well div elements. 


# 中文版本FCC
Bootstrap：在 Bootstrap Wells 中添加元素

现在我们的每一行的每一列都已经有了 div元素。这已经足够了，现在让我们添加 button元素吧。

在每一个 well div元素下放置三个 button元素。

```html
<div class="container-fluid">
  <h3 class="text-primary text-center">jQuery Playground</h3>

      </div>
    </div>
    <div class="col-xs-6">
      <div class="well">



      </div>
    </div>
  </div>
</div>
```


# Answer
```html
<div class="container-fluid">
  <h3 class="text-primary text-center">jQuery Playground</h3>
  <div class="row">
    <div class="col-xs-6">
      <div class="well">
        <button></button>
        <button></button>
        <button></button>
      </div>
    </div>
    <div class="col-xs-6">
      <div class="well">
        <button></button>
        <button></button>
        <button></button>
      </div>
    </div>
  </div>
</div>
```
